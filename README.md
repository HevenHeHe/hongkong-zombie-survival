# Hong Kong Zombie Survival

A 3D survival game inspired by Hong Kong zombie cinema (僵尸片风格).

## Genre
- Survival / Resource Management
- Action RPG

## Art Style
- Low Poly + Cartoon blend
- Hong Kong street aesthetic

## Tech Stack
- Three.js (3D rendering)
- Cannon.js (Physics)
- React (UI)
