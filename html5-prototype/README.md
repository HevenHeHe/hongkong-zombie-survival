# Hong Kong Zombie Survival - HTML5 Prototype

## Overview
A lightweight HTML5 survival game prototype built with Three.js and Cannon.js. 
Designed for CPI testing with low-poly aesthetic inspired by Hong Kong zombie cinema.

## Tech Stack
- **Engine**: Three.js r158 (WebGL)
- **Physics**: Cannon-ES
- **Style**: Low Poly + Neon Cyberpunk UI

## Controls
- **WASD**: Move player
- **Mouse**: Look around (pointer lock)
- **SPACE**: Jump
- **Survival Goal**: Avoid zombies, survive as long as possible

## Game Features
- 3D Hong Kong street environment with buildings
- Zombie AI that chases the player
- Health system
- Score based on survival time
- Day/night progression
